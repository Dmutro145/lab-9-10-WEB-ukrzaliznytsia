# lab-9-10-WEB-ukrzaliznytsia
##  Додаткові покращення
- Сортування рейсів (за часом, місцями, тривалістю)
- Сторінка деталей потяга
- Статистика місць у схемі вагона
- Видалення окремого бронювання
- Лічильник рейсів на головній сторінці
- Кнопка "Прокрутити вгору"
- Темна тема
- Поле кількість пасажирів у формі
- Прогрес-бар зайнятості місць на картці
- Підтвердження перед очищенням бронювань
- Кнопка скидання вибору місць
- Лічильник бронювань на кнопці "Мої бронювання"
- Підсвітка поля пошуку при фокусі
